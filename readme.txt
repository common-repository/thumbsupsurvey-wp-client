== Description ==
The plugin provides the necessary shortcodes and a builder to easily add your surveys in any page or post, including all possible options, such as web or form, even for SalesForce embedded chat.

For more information, visit [https://thumbsupsurvey.com/](https://thumbsupsurvey.com/).

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/thumbsupsurvey-wp-client` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use one of the three available shortcodes [thumbsupsurvey-web], [thumbsupsurvey-form], [thumbsupsurvey-salesforce] within a page or a post, using the necessary license key.
4. Use the helper to set up the shortcode with all the possible options.

For more information, visit our Knowledgebase: [https://thumbsupsurvey.com/using-the-survey-with-wordpress/](https://thumbsupsurvey.com/using-the-survey-with-wordpress/).

== Third-Party Service Integration ==
This plugin relies on a third-party service, [ThumbsUpSurvey.com](https://thumbsupsurvey.com), to provide the survey functionality. The plugin works as a helper for setting up the correct parameters and initiates the call to the main script coming from [ThumbsUpSurvey.com](https://thumbsupsurvey.com)
